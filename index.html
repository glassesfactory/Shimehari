• Shimehari is Python Middle Framework for web.

    Shimehari is Yopparai.

    $ cd your_proj_directory
$ shimehari create your_app_name
$ shimehari drink
$ * Shimehari GKGK!
$ * Running on http://127.0.0.1:5959/

    酔っててもインストールできるますよ

    $ pip install Shimehari
    もしくは
    $ git clone git@github.com:glassesfactory/Shimehari.git
$ python setup.py install
    とか
    $ wget http://pypi.python.org/packages/source/S/Shimehari/Shimehari-0.1.5.tar.gz
$ tar zxvf Shimehari-0.1.5.tar.gz
$ cd Shimehari-0.1.5
$ python setup.py install

    気になったら
