=============
Shimehari
=============

Python Middle Framework for web.

inspired Flask:: http://flask.pocoo.org

What is Shimehari
====================

Shimehari (しめはり)は Python 製 Web フレームワークです。

フルスタックだとゴリゴリすぎるけどマイクロフレームワークだと
ちょっと規模が大きいかな… という時に活躍したいミドルフレームワークを自称しています。

Shimehari がやってくれること
========================================

Shimehari はルーティングとコントローラーの枠組だけ提供します。
Form ヘルパーや ORM 、Scaffold といった高度な機能は提供しません。

代わりに Rails のように様々なコマンドラインツールを用意しています。
たいていの作業はコマンドラインで完結します。

Shimehari の由来
===================

我々の大好きなお酒「〆張鶴」から名前をいただいています。
「〆張鶴」のように、すっきり爽やか、しかし芯のあるしっかりしたフレームワークにしたいと思っています。
