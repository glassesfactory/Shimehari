Welcome to Shimehari
========================

Welcome to Shimehari's documentation!

Shimehari(しめはり) は Python 製 Web フレームワークです。

フルスタックだと扱いがヘヴィだけどマイクロフレームワークでは
ちょっとものたりないかな… という時に活躍したいと考えている
自称ミドルフレームワークです。

まずは :ref:`installation` と :ref:`quickstart` を読んでみてください。
